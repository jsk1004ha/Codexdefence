# 오비탈 코어: 아이들 타워 디펜스

브라우저에서 실행되는 데이터 중심 방치형 타워 디펜스입니다.

## 실행 방법

```bash
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```

## 현재 구현 범위 (강화됨)

- 중앙 코어 자동 전투 + 웨이브 + 보스 페이즈 패턴 + 정예 접두어 20종 효과 반영
- 코어 카테고리/히든 코어 포함 대량 코어 데이터 + 카테고리별 공격 패턴 분화
- 궤도 모듈 20종의 개별 효과(레이저/실드/중력/냉각/드론/미사일/처형/정화/오메가 복제 등)
- 궁극기 15종 개별 발동 로직 + 각성 방향 데이터/런 중 선택(자동 배정)
- 런 중 업그레이드 카드 선택, 재굴림, 위험 보상(히든 단서 점수 연동)
- 영구 성장/연구소/유물/마스터리/프레스티지/도전 모드 실효 규칙 반영
- 미션 보상 수령, 칭호 해금, 히든 단서 누적, 히든 보스 조우 플래그
- localStorage 저장/불러오기 + 오프라인 보상

## 조작

- `전투 시작`: 런 시작
- `Space`: 궁극기 사용
- `r`: 즉시 귀환 정산
- 상단 메뉴 버튼: 각 시스템 화면 전환

## 프로젝트 구조

- `index.html`: 캔버스 + HUD + 패널 레이아웃
- `styles.css`: SF 글래스모피즘 UI 스타일
- `app.js`: 데이터 정의, 게임 루프, 전투/성장/저장/화면 로직

## 주의

- 외부 이미지 없이 Canvas 절차적 그래픽으로 렌더링합니다.
- 대규모 콘텐츠는 데이터 배열 기반으로 확장 가능하게 구성했습니다.
