# 오비탈 코어: 아이들 타워 디펜스

브라우저에서 실행되는 데이터 중심 방치형 타워 디펜스입니다.

## 실행 방법

```bash
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080 접속
```

## 현재 구현 범위 (UI/UX 2차 개편)

- **SF 글래스모피즘 전면 UI**: 상단 전투 HUD, 좌측 내비게이션 레일, 우측 정보 패널로 재구성.
- **메인 전투 뷰 개선**: 전투/귀환/궁극기 동선 단순화, 실시간 모듈 상태 카드 제공.
- **Upgrade & Mastery Center 신설**: 코인/에너지를 소모하는 영구 업그레이드(공격/체력/수익/궁극기 충전/모듈 슬롯).
- **Mission & Rewards Dashboard 개선**: 진행률 바, 보상 흐름, 카드형 미션 UX.
- **골드 소모처 확장**: 영구 성장 업그레이드 구매로 누적 코인(골드) 활용 가능.
- **반응형 + 모바일 UI 지원**: 모바일 하단 퀵컨트롤(시작/궁극기/귀환/패널)과 좁은 화면 레이아웃 대응.
- 중앙 코어 자동 전투 + 웨이브 + 보스 페이즈 패턴 + 정예 접두어 20종 효과 반영.
- 코어 카테고리/히든 코어 포함 대량 코어 데이터 + 카테고리별 공격 패턴 분화.
- 궤도 모듈 20종의 개별 효과(레이저/실드/중력/냉각/드론/미사일/처형/정화/오메가 복제 등).
- 궁극기 15종 개별 발동 로직 + 각성 방향 데이터/런 중 선택(자동 배정).
- 런 중 업그레이드 카드 선택, 재굴림, 위험 보상(히든 단서 점수 연동).
- 영구 성장/연구소/유물/마스터리/프레스티지/도전 모드 실효 규칙 반영.
- localStorage 저장/불러오기 + 오프라인 보상.

## 조작

- `Start Run`: 런 시작
- `Space`: 궁극기 사용
- `r`: 즉시 귀환 정산
- 모바일 하단 버튼: 시작/궁극기/귀환/패널 접근
- 좌측 메뉴 버튼: 각 시스템 화면 전환

## 프로젝트 구조

- `index.html`: 캔버스 + HUD + 네비게이션/패널/모바일 컨트롤 레이아웃
- `styles.css`: SF 글래스모피즘 + 반응형/모바일 스타일
- `app.js`: 데이터 정의, 게임 루프, 전투/성장/저장/화면 로직

## 주의

- 외부 이미지 없이 Canvas 절차적 그래픽으로 렌더링합니다.
- 대규모 콘텐츠는 데이터 배열 기반으로 확장 가능하게 구성했습니다.
